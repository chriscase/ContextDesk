# Non-goals

Explicitly out of scope unless a future major version reopens them:

1. **Code-editing agent** — apply_patch, autonomous multi-file refactors, default shell tools  
2. **Silent remote writes** — Confluence/wiki/API mutations without modal confirmation  
3. **Unrestricted tools** — free-form `curl`, arbitrary shell, whole-home filesystem index  
4. **Employer-specific code** — private gateway hostnames, internal model catalogs, company logos in core  
5. **Replacing personal PKM apps** — we are eng-ops knowledge, not full Obsidian clone  
6. **Full collab suite** — Google-Docs-grade CRDT editing / presence is not MVP  
7. **MCP marketplace auto-install** — opt-in few servers only  
8. **Embedding a full third-party agent runtime** — use chat APIs + local ContextDesk tools  

When in doubt, prefer smaller connectors and stronger citations over more automation.
